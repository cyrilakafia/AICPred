StormPred

StormPred is a web application that uses machine learning to predict small molecules that can inhibit the TLR4’s pro-inflammatory role during the COVID-19 cytokine storm. The models were trained on 195,980 experimentally-tested TLR4 activators and inhibitors. The web app offers predictions using the Decision Tree Classifier model, which achieved a balanced accuracy of 0.94 and an AUROC score of 0.94 and the XGBoost model, which achieved a balanced accuracy of 0.96 and AUCROC score of 0.99.
